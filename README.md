# 💤 LazyVim Configuration

This repository contains my personalized configuration for [LazyVim](https://github.com/LazyVim/LazyVim), a Neovim starter template.

## Installation

Follow the [LazyVim documentation](https://lazyvim.github.io/installation) to set up LazyVim. Once installed, clone this repository into your Neovim configuration directory:

```bash
git clone git@github.com:Ororura/nvim.git ~/.config/nvim
```

## Features

- Custom key mappings for improved productivity.
- Additional plugins tailored for my workflow.
- Optimized settings for a smooth Neovim experience.
